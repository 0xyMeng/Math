# 非线性的改善与利用


- 从线性角度出发

从线性的角度考虑去避免自振，可以加相角超前，使得线性部分幅相曲线不和负倒描述函数曲线不相交。

从局部反馈的角度，加一个局部增益非常大的深度负反馈，可以有效减弱非线性的影响。


- 改变非线性特性

此外还可以去改变非线性特性，比如有死区。

- 非线性特性利用

人为引入非线性。
