# 分析学(analysis)

以微积分方法为基本工具，以函数（映射、关系等更丰富的内涵）为主要研究对象，以极限为基本思想的众多数学经典分支及其现代拓展的统称，简称分析。

- 狭义的分析学——数学分析：以微分学、积分学、级数论、实数理论为其基本内容。
- 广义的分析学：极限的概念不仅是微积分的核心，也是许多其他学科的重要思想。随着微积分出现的一些新的数学分支如微分方程、函数论、变分法、泛函分析等，统称为广义的分析学。
